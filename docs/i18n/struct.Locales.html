Struct Locales

pub struct Locales { /* private fields */ }
Expand description

A thread-safe container for all loaded localization data.

It manages multiple Locale instances, keyed by language identifier, and provides a unified interface for querying translations. It also handles fallback logic to a default language if a translation is missing.

pub fn query( &self, lang: &LanguageIdentifier, query: &Query<'_>, ) -> Result<Message, Vec<FluentError>>

Queries for a message in a specific language, applying fallback logic if the language is not found.

It first attempts to find the Locale for the requested language. If the entire Locale is missing, it will automatically retry the query using the configured fallback language.
